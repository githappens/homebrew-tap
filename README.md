# githappens/homebrew-tap

Personal Homebrew tap for tools published by [@githappens](https://github.com/githappens).

## Install

```bash
brew tap githappens/tap
```

## Formulae

- [`busybee`](https://github.com/githappens/busybee) — queued runner for
  resource-heavy tasks with a live CPU + queue TUI monitor.
  ```bash
  brew install githappens/tap/busybee
  ```
